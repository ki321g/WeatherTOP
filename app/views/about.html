About WeatherTop APP

The WeatherTop companion Web APP was developed in by WeatherTop Inc to allow meteorologists who use there modular weather station called the WeatherTop 1000 to record the readings.

The low-cost device takes a set of meteorological readings and displays these on a simple LCD display. WeatherTop App is a Web companion application for the device.
